# Scroll Video

A WordPress plugin that synchronizes video playback with page scroll using GSAP ScrollTrigger.

## Installation

1. Upload the plugin to `/wp-content/plugins/scrollvideo/`
2. Activate the plugin in WordPress admin

## Usage

Use the `[scrollvideo]` shortcode with a video source:

```
[scrollvideo src="https://example.com/video.mp4"]
```

### Parameters

- `src` (required): URL to the video file
